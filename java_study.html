javascript 공부 시작!

1. 기본 타입

  • 타입은 typeof 를 통해 알 수 있다.
  • 숫자
  • 문자열

    - 문자열은 더하는 것이 가능하다.(숫자 입력시 숫자를 문자열로

  • 불리언
  • 심볼

    - 심볼은 유일하고 변경할 수 없는 타입으로, 객체의 프로퍼티를 위한 식별자로 사용할 수 있다.

    이게 무슨 소리일까,,,

  • undefined

    - null은 object 타입, 값이 정해지지 않음

    - undefined는 타입이 정해지지 않음. 초기화 되지 않은 변수나 존재하지않는 값일 경우

  • 객체

    - 여러 프로퍼티나 메소드를 같은 이름으로 묶어 놓는 집합체

2. 타입 변환

  • 묵시적 타입 변환

    - 자동으로 타입 변환

  • 명시적 타입 변환

    - 변환을 위한 함수 사용

  • 숫자를 문자열, 불리언 값을 문자열, 날짜를 문자열이나 숫자, 문자열을 숫자로 바꿔주는 함수 존재 -> 필요하면 찾아보기

3. 연산자

  • 대부분 파이썬, c, c++ 과 비슷함.. 다른것만 적음!!
  • 산술 연산자
  • 대입 연산자
  • 증감 연산자
  • 비교 연산자

    - === : 값과 타입이 같으면 참을 반환

    - !== : 값이 같지 않거나 타입이 다르면 참을 반환

  • 논리 연산자
  • 비트 연산자

    - ^ : 대응되는 비트가 다르면 1 반환(XOR)

    - >>> : 지정 수만큼 비트를 전부 오른쪽으로 이동시키며, 새로운 비트는 0

  • 기타 연산자

    - '+'연산자로 문자열 결합 수행

    - instanceof : 피연산자인 객체가 특정 객체의 인스턴스인지 아닌지 판별

    - void : 항상 undefined 값 반환

4. 조건문

  • - if문, if/else 문 , if/ else if/ else 문, switch 문
  • - python, c, c++ 의 조건문과 똑같음!!

5. 반복문

  • - while, do/while, for, for/in, for/of
  • - for/of문 제외하고는 python, c, c++ 과 같음
  • - for/of문 : 반복할 수 있는 객체를 순회할 수 있도록 해줌.

6. 기타 제어문

  • - label문 : 프로그램내의 특정 영역을 식별할 수 있게 해주는 식별자입니다.
  • - continue, break : python, c, c++ 과 동일

7. 배열

  • - 생성 : 배열 리터럴 [] 사용, Array 객체 생성자 사용, new 연산자 사용
  • - 요소 추가 : Push 메소드 사용, length 프로퍼티 사용(arr[arr.length]), 특정 인덱스 지정
  • - 배열 순회 : 반복문 for문을 써서 모든 요소에 접근 가능 -> forEach, callback 함수를 사용하는 것이 더 효과적

8. 함수

  • - 기본적인 사용법은 python, c, c++ 과 같음
  • - javascript에서의 함수는 값으로도 사용가능하다.
  • - 함수 호이스팅 : 함수안에서 선언된 모든 변수는 함수 전체에 걸쳐 유효하다.
  • - 함수를 호출할 때 함수의 정의보다 적은 수의 인수가 전달되어도 오류가 발생하지 않고 undefined 값을 설정한다.
  • - arguments 객체 : 매개변수 대신 사용, 배열의 형태
  • - default 매개변수 : 함수를 호출할 때 명시된 인수를 전달하지 않았을 경우에 사용할 것
  • - rest(나머지) 매개변수 : 생략접두사 (...) 을 이용하여 특정 위치 인수부터 마지막 인수까지 한번에 지정 가능

9. 객체

  • - 객체는 실생활에서 우리가 인식할 수 있는 사물
  • - 객체의 생성 : 리터럴 표기 이용(객체 = {이름:값}), 생성자 함수 이용, Object.create() 메소드 이용
